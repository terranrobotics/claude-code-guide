Your Terminal

We recommend iTerm2 — download free at iterm2.com. It's a better version of the Terminal app that comes with your Mac.

⌘ TNew tab
⌘ WClose tab
⌘ DSplit screen side by side
⌘ Shift DSplit screen top/bottom
⌘ ] / ⌘ [Switch between splits
⌘ ← / ⌘ →Switch tabs

You can run multiple Claude sessions at the same time — one per split or tab. Great for working on two things in parallel.

Getting Around

Your terminal is always "in" a folder. Think of it like having a Finder window open — commands happen in whatever folder you're in.

pwdWhat folder am I in?
lsWhat's in this folder?
cd foldernameOpen a folder
cd ..Go back up one folder
cd ~Go to your home folder
open .Open this folder in Finder
code .Open this folder in VS Code

~ means your home folder: /Users/yourname

Starting Claude

# go to your project folder first
cd ~/src/terran
claude

Claude needs to be started from your project folder so it knows what you're working on.

claude -cPick up where you left off last time
claude -rChoose an older session to go back to

Skip the cd Step

Paste this once and open a new tab. Now typing claude anywhere always starts in the right folder:

echo 'alias claude="cd ~/src/terran && command claude"' >> ~/.zshrc

Talking to Claude

Just type normally. You don't need special commands — Claude understands plain English.

  • "Read the file src/main.py"
  • "Find all files that mention HubSpot"
  • "What does this function do?"
  • "What changed since yesterday?"
  • "Draft an email to Bob from accounting about the schedule"
  • "Search my calendar for meetings this week"

Claude reads and edits your actual files — you don't need to copy-paste anything.

Run Commands from Claude

Type ! before any terminal command to run it without leaving your session:

! git status
! ls ~/src/terran

Commands & Skills

Type these inside a Claude session. They all start with /

/helpSee everything Claude can do
/clearStart a fresh conversation
/compactIf Claude starts acting odd, this resets its memory
/resumeGo back to an older conversation
/costSee how much you've used
/doctorCheck if everything is working
/modelSwitch to a different AI model

Your team may also have custom skills — type / and scroll to see what's available.

Good Habits

  • Review before approving. When Claude wants to run a command or edit a file, it asks permission. Read what it's about to do before pressing Enter.
  • Teach Claude as you go. If Claude figures something out, say "remember that." It saves the lesson for next time.
  • Use Esc to back out. If Claude suggests something you don't want, press Esc to decline and try a different approach.
  • Short sessions are better. Don't let one conversation run for hours. Start fresh when you switch topics.

If Something Goes Wrong

  • Claude is stuck or looping — press Ctrl+C, then try /compact
  • Screen is a mess — press ⌘ K to clear it
  • Claude forgot what we were doing — long conversations lose context. Start a new session.
  • Nothing is working — type /doctor to check your setup
  • Command not found — make sure you're in the right folder (cd ~/src/terran)
  • Really stuck — close the tab, open a new one, start fresh
